## Test Environment
  * Windows 10
  * Visual Studio 2017
  * Surface Book 2
  
## Library Dependencies
  * OpenCV
  * OpenGL

## Keyboard Commands
  * **i key**: reset the main camera
  * **w key**: move up
  * **s key**: move down
  * **Up arrow**: move forward
  * **Down arrow**: move backward
  * **Left arrow**: move left
  * **Right arrow**: move right
  * **q key**: exit
